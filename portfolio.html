Projects

Project 01: PySnake

This Snake game was built using Pygame and follows an object-oriented approach, featuring classes for the snake, food, and game logic. The classic arcade-style game involves controlling a growing snake on a board while avoiding obstacles and edges. Building this project improved my skills in Python, OOP, and game development, teaching me to design classes and methods, manage graphics and display updates, handle events and game states, and write efficient code.

Project 02: Color Cluster

ColorCluster is a Python tool I built that extracts the dominant colors from an image and presents them as a color palette. The tool was built using OpenCV and Scikit-learn libraries to cluster pixels and sort them by frequency. This project was a great way for me to learn about image processing and unsupervised machine learning algorithms. I also learned how to work with OpenCV and NumPy, and how to apply KMeans clustering to pixel data.

Project 03: Blender

Blender.Co is an innovative blockchain platform designed to provide users with a secure and confidential way to calculate their crypto credit score. Built with a user-friendly interface and interactive app, Blender.Co assesses a user's creditworthiness based on multiple factors and delivers the results quickly and efficiently. This project taught me how to design and develop a user interface that is simple to navigate and intuitive to use.
